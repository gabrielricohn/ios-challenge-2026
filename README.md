## Getting Started

This is an example of how you may give instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* Homebrew

Some of the tools we use we recommend installing using Homebrew 

  ```sh
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  ```

* Ruby

We recommends having a Ruby Version Manager installed becase we will be working with some Ruby Gems. Here are some alternatives:

* [rbenv][rbenvUrl]
* [rvm][rvmUrl]
* [asdf][asdfUrl]

Most users work with rbenv.

Recommended Ruby Version: ruby 3.1.0

* Install Bundler

The Ruby Gems we need for the project are declared in a file called Gemfile and this is the command you need to run to install those dependencies:

```sh
    gem install bundler
    ...
    bundle install
```

* Tuist 

Tuist is a command-line tool (CLI) that facilitates the generation, maintenance, and interaction with Xcode projects. [Tuist][TuistUrl]

Install TUIST using the following steps:

1. install mise CLI(is a development environment setup tool.) [Mise][MiseUrl]

```sh
curl https://mise.run | sh
~/.local/bin/mise --version
```

2. active mise

```sh
echo 'eval "$(~/.local/bin/mise activate zsh)"' >> ~/.zshrc
```

3. Run this command to add the environment variable that will prevent mise from automatically installing Ruby and Node.

```sh
echo 'export MISE_DISABLE_TOOLS="ruby,nodejs"' >> ~/.zshrc
```

4. Close and reopen your terminal to apply the changes.

5. Execute the following command in the root of the project. This command will install every tool defined in the mise.toml file. Install Tuist

```sh
mise install tuist@4.148.1
```

6. If tuist generate doesn't work at the first time please execute this command

```sh
mise install tuist@4.148.1
#OR
mise use tuist@4.148.1
```

* Arkana

At this point you need to install Arkana create the .env files. [Arkana][ArkanaUrl]

```sh
nano .env
nano .env.development
```

Then execute the bundler

```sh
bundle install #only if it was not executed before
bundle exec arkana -c .arkana.yml -f Development
```

### Installation

1. Clone/Fork the repo
   ```sh
   git clone git@github.com:applaudo-cc/ios-challenge-2026.git
   ```
2. Install SPM packages
   ```sh
   tuist fetch
   ```
3. Generate XCode.proj
   ```sh
   tuist generate
   ```
